# Questionlets

A minimal Quarto website configured for automatic deployment to GitHub Pages.

## Preview locally

```bash
quarto preview
```

## Publish

Push the repository to GitHub, then select **Settings → Pages → Source → GitHub Actions**. Each push to `main` will build and deploy the site.

Before publishing, replace `YOUR-GITHUB-USERNAME` in `_quarto.yml` and `index.qmd` with your GitHub username.
